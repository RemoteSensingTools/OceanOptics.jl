Pure (sea)water

OceanOptics.PureWater — Type
struct PureWater{FT, A<:AbstractPureWaterAbsorptionModel, S<:AbstractPureWaterScatteringModel} <: AbstractAbsorbingScatterer{FT}

Pure (sea)water as an optical constituent.

Fields

  • absorption_model::AbstractPureWaterAbsorptionModel: Absorption-model choice (tag)

  • scattering_model::AbstractPureWaterScatteringModel: Scattering-model choice (tag)

  • temperature::Any: Temperature [°C]

  • salinity::Any: Salinity [PSU] (Practical Salinity Unit)

  • depolarization::Any: Depolarization ratio for the Rayleigh-type scattering phase function

Examples

# Default: piecewise absorption, Zhang-Hu scattering, S=35 PSU, T=20 °C
water = PureWater{Float64}()

# Legacy Pope-Fry + Morel combination
water = PureWater{Float64}(
    absorption_model = PopeFry1997(),
    scattering_model = Morel1974(),
)

# Cold, freshwater Arctic case
water = PureWater{Float64}(temperature = 2.0, salinity = 2.0)

CDOM

OceanOptics.CDOM — Type
struct CDOM{FT, M<:AbstractCDOMModel} <: AbstractAbsorber{FT}

Coloured Dissolved Organic Matter (CDOM) as a pure absorber.

Fields

  • model::AbstractCDOMModel: Parameterization-model tag (e.g. BricaudMorelPrieur1981())

  • a_ref::Any: Reference absorption a_g(λ_ref) [1/m]

  • λ_ref::Any: Reference wavelength [nm] at which a_ref is anchored

  • slope::Any: Spectral slope S [1/nm]

Examples

# Open-ocean default: a_g(440 nm) = 0.03 m⁻¹, slope 0.014 nm⁻¹
cdom = CDOM{Float64}()

# Coastal / humic-rich case
cdom_coastal = CDOM{Float64}(a_ref = 0.5, λ_ref = 440.0, slope = 0.017)

# Evaluate
absorption(cdom, 440.0)   # ≈ 0.03
absorption(cdom, 500.0)   # exponentially smaller

OceanOptics.BricaudMorelPrieur1981 — Type
struct BricaudMorelPrieur1981 <: AbstractCDOMModel

Bricaud, Morel & Prieur (1981) exponential-decay CDOM absorption, a_g(λ) = a_g(λ_ref) · exp(-slope · (λ - λ_ref)). Currently the only parameterization shipped; future additions (e.g. separate slopes for the UV and visible hinge regions) can subtype AbstractCDOMModel without altering CDOM itself.

Phytoplankton

OceanOptics.Phytoplankton — Type
struct Phytoplankton{FT, M<:AbstractPhytoplanktonModel, P<:AbstractOceanPhaseFunction{FT}} <: AbstractAbsorbingScatterer{FT}

Phytoplankton (living algal cells) as an absorbing scatterer.

Chl is the near-surface chlorophyll-a concentration [mg/m³]. The model tag M selects the absorption parameterization; scattering currently uses Morel & Maritorena (2001) irrespective of M, and the phase function defaults to FournierForandPhase with backscatter fraction 0.01 (typical open-ocean living phytoplankton).

Fields

  • model::AbstractPhytoplanktonModel: Parameterization-model tag (e.g. Bricaud1998())

  • Chl::Any: Chlorophyll-a concentration [mg/m³]

  • phase::AbstractOceanPhaseFunction{FT} where FT: Particulate-scattering phase function

Examples

# Open-ocean Case-1 water at Chl = 0.5 mg/m³
phyto = Phytoplankton{Float64, Bricaud1998}(Chl = 0.5)

# Custom backscatter fraction (coastal, larger cells)
phyto = Phytoplankton{Float64, Bricaud1998}(Chl = 2.0, B = 0.015)
source
OceanOptics.Bricaud1998 — Type
struct Bricaud1998 <: AbstractPhytoplanktonModel

Bricaud et al. (1998, JGR 103, 31033) power-law phytoplankton absorption, extended to UV by Morrison & Nelson (2004) and Vasilkov et al. (2005). Bundled as data/bricaud_1998_A.csv and data/bricaud_1998_E.csv, covering 300–1000 nm at 2 nm resolution.

source

Non-algal particles

OceanOptics.NonAlgalParticles — Type
struct NonAlgalParticles{FT, M<:AbstractNAPModel, P<:AbstractOceanPhaseFunction{FT}} <: AbstractAbsorbingScatterer{FT}

Non-algal particulate matter (detritus + minerals) as an absorbing scatterer.

Fields

  • model::AbstractNAPModel: Parameterization-model tag (e.g. BabinBricaud2003() or FellMineral())

  • a_ref::Any: Reference absorption a_nap(λ_ref) [1/m] (BabinBricaud2003)

  • b_ref::Any: Reference scattering b_nap(λ_ref) [1/m] (BabinBricaud2003)

  • λ_ref::Any: Reference wavelength [nm]

  • slope::Any: Exponential absorption slope [1/nm] (BabinBricaud2003)

  • c_ref::Any: Beam attenuation at 440 nm [1/m] (FellMineral)

  • ω_nap::Any: Single-scattering albedo at 440 nm (FellMineral)

  • phase::AbstractOceanPhaseFunction{FT} where FT: Particulate-scattering phase function

Examples

# Babin et al. (2003) default: a_nap(443) = 0.05 m⁻¹, S = 0.011 1/nm
nap = NonAlgalParticles{Float64}()

# Mineral-dominated case per Fell (1997)
nap = NonAlgalParticles{Float64, FellMineral}(
    c_ref  = 0.40,   # beam attenuation at 440 nm
    ω_nap  = 0.96,   # single-scattering albedo
)

Backscattering coefficient b_b(λ) [1/m] — the integral of the volume scattering function over the back hemisphere. Equals b(λ) * B where B is the backscatter fraction determined by the phase function.
