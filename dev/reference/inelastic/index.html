Inelastic processes

Fluorescence, water Raman, and CDOM fluorescence packaged as kernels consumed by a two-pass RT solver (Fell 1997 §2.2.9).

Spectral redistribution

OceanOptics.AbstractSpectralRedistribution — Type
abstract type AbstractSpectralRedistribution{FT<:AbstractFloat}

Spectral redistribution f^I(λ', λ) [1/nm] for an inelastic process: the probability density of emission wavelength λ given excitation at λ'. Concrete subtypes implement redistribution.

OceanOptics.GaussianEmission — Type
struct GaussianEmission{FT<:AbstractFloat} <: AbstractSpectralRedistribution{FT<:AbstractFloat}

Gaussian emission band in wavelength space centered at a fixed wavelength, independent of excitation. Canonical example: chlorophyll-a fluorescence,

GaussianEmission{Float64}(center_nm = 685.0, σ_nm = 10.6)

which matches the 685 nm peak and σ = 10.6 nm (FWHM ≈ 24.9 nm) used in Fell (1997) §2.2.9 and Gordon (1979) for the oceanic chlorophyll-fluorescence profile.

Fields

  • center_nm::AbstractFloat: Centre of the emission Gaussian [nm]

  • σ_nm::AbstractFloat: Standard deviation of the emission Gaussian [nm]

OceanOptics.WavenumberShiftEmission — Type
struct WavenumberShiftEmission{FT<:AbstractFloat} <: AbstractSpectralRedistribution{FT<:AbstractFloat}

Gaussian emission band in wavenumber space, centered at a fixed offset from the excitation wavenumber. Used for water Raman scattering, where emission wavenumber ν = ν' − Δν with Δν ≈ 3400 cm⁻¹ (Haltrin & Kattawar 1991/1993).

Construction:

# Water Raman: 3400 cm⁻¹ Stokes shift, 85 cm⁻¹ Gaussian σ
WavenumberShiftEmission{Float64}(shift_cm_inv = 3400.0, σ_cm_inv = 85.0)

Fields

  • shift_cm_inv::AbstractFloat: Stokes shift Δν = ν_excitation − ν_emission [1/cm]

  • σ_cm_inv::AbstractFloat: Gaussian standard deviation in wavenumber [1/cm]

Mathematical note

The stored Gaussian is normalized in wavenumber (∫ g(ν) dν = 1); redistribution applies the Jacobian |dν/dλ| = 10⁷/λ² so the emitted density f(λ', λ) is also normalized in wavelength, satisfying ∫ f dλ = 1.

OceanOptics.IsotropicFluorescence — Type
struct IsotropicFluorescence{FT, A<:AbstractAbsorber{FT}, E<:AbstractSpectralRedistribution{FT}} <: AbstractOceanInelasticProcess{FT}

Isotropic fluorescence from an AbstractAbsorber with a configurable spectral redistribution and quantum yield.

Fields

  • absorber::AbstractAbsorber: Absorbing constituent whose absorbed photons drive emission

  • emission::AbstractSpectralRedistribution{FT} where FT: Spectral redistribution f(λ', λ) [1/nm]

  • quantum_yield::Any: Quantum yield φ (dimensionless, typically 0.001–0.01 for Chl-a)

  • excitation_range::Tuple{FT, FT} where FT: Excitation-wavelength integration window [nm]

Examples

phyto = Phytoplankton{Float64, Bricaud1998}(Chl = 0.5)
sif   = chlorophyll_fluorescence(phyto)                   # default config
sif   = IsotropicFluorescence{Float64}(phyto;
            emission        = GaussianEmission{Float64}(center_nm = 685.0,
                                                        σ_nm      = 10.6),
            quantum_yield   = 0.003,
            excitation_range = (400.0, 700.0))

OceanOptics.CDOMFluorescence — Type
struct CDOMFluorescence{FT, E<:AbstractSpectralRedistribution{FT}} <: AbstractOceanInelasticProcess{FT}

CDOM fluorescence as an isotropic inelastic process. Structurally equivalent to IsotropicFluorescence specialized on a CDOM absorber, but carried as a distinct type so solvers can dispatch separately and users can read model intent directly from the type.

Typical excitation/emission footprint per Coble (1996, 2007) and Bricaud-Morel-Prieur (1981): broad CDOM emission around 420–450 nm for UV excitation (300–380 nm), with quantum yield 1–2 % when integrated over the band. Fell (1997) uses a simple Gaussian model.

Fields

  • cdom::CDOM: CDOM absorber driving the fluorescence

  • emission::AbstractSpectralRedistribution{FT} where FT: Spectral redistribution f(λ', λ) [1/nm]

  • quantum_yield::Any: Quantum yield φ_g (dimensionless, band-integrated)

  • excitation_range::Tuple{FT, FT} where FT: Excitation-wavelength integration window [nm]

OceanOptics.WaterRaman — Type
struct WaterRaman{FT<:AbstractFloat} <: AbstractOceanInelasticProcess{FT<:AbstractFloat}

Water Raman scattering as an inelastic process. Emission spectrum is a Gaussian in wavenumber centred at ν' − shift_cm_inv; cross-section follows a Haltrin-Kattawar power law.

Fields

  • shift_cm_inv::AbstractFloat: Stokes shift between excitation and emission wavenumbers [1/cm]

  • σ_cm_inv::AbstractFloat: Gaussian standard deviation in wavenumber space [1/cm]

  • cross_section_ref::AbstractFloat: Raman cross-section b_w^R at the reference wavelength [1/m]

  • cross_section_λ_ref::AbstractFloat: Reference wavelength for the cross-section power law [nm]

  • cross_section_n::AbstractFloat: Exponent n in b_w^R(λ') = b_ref · (λ_ref / λ')^n

  • depolarization::AbstractFloat: Cabannes depolarization ratio ρ_Cab (≈ 0.14 for liquid water)

  • excitation_range::Tuple{FT, FT} where FT<:AbstractFloat: Excitation-wavelength integration window [nm]

Construction

# Default: Haltrin-Kattawar/Bartlett parameterization
WaterRaman{Float64}()

# Customize (e.g. with Bartlett salt- and temperature-aware values)
WaterRaman{Float64}(
    shift_cm_inv       = 3400.0,
    σ_cm_inv           = 85.0,
    cross_section_ref  = 2.7e-4,    # 1/m at reference wavelength
    cross_section_λ_ref = 488.0,    # nm
    cross_section_n     = 5.5,
    depolarization      = 0.14,     # Cabannes ρ for liquid water
    excitation_range    = (350.0, 700.0),
)

Trait interface

OceanOptics.AbstractOceanInelasticProcess — Type
abstract type AbstractOceanInelasticProcess{FT<:AbstractFloat}

Root of the inelastic-source hierarchy (DESIGN §4.4 / §7.1). Concrete subtypes — IsotropicFluorescence, WaterRaman, CDOMFluorescence — describe one excitation-to-emission channel apiece and implement the Fell (1997) factorization primitives excitation_absorption(p, λ'), emission(p, λ', λ), excitation_range(p), and is_isotropic(p).

source
OceanOptics.excitation_absorption — Function
excitation_absorption(p::AbstractOceanInelasticProcess, λ_prime) -> FT

Absorption-side coefficient a^I(λ') [1/m] in Fell's factorization b^I(λ', λ) = a^I(λ') · f^I(λ', λ).

For fluorescence this is the quantum-yield-weighted absorption of the fluorescing material (φ · a_absorber(λ')); for Raman it is the Raman-scattering cross-section b_w^R(λ') taken in the "absorption role" of the inelastic-source operator.

source
OceanOptics.emission — Function
emission(p::AbstractOceanInelasticProcess, λ_prime, λ) -> FT

Spectral redistribution density f^I(λ', λ) [1/nm], normalized so ∫ f(λ', λ) dλ = 1 for every λ'.

source
OceanOptics.excitation_range — Function
excitation_range(p::AbstractOceanInelasticProcess) -> (λ_min, λ_max)

Wavelength window [nm] over which the excitation integral should be taken. Outside this window excitation_absorption is understood to be negligible, allowing integrators to skip unnecessary grid points.

source
OceanOptics.is_isotropic — Function
is_isotropic(p::AbstractOceanInelasticProcess) -> Bool

Whether the process emits isotropically in angle. Isotropic emission projects onto the m = 0 azimuthal Fourier mode only, which the Fell machinery exploits to halve the work. Fluorescence of solution-phase molecules (chlorophyll, CDOM, most pigments) is isotropic; polarization-preserving Raman is not.

source
is_isotropic(p::WaterRaman) -> Bool

Water Raman is Cabannes-depolarized: the re-emitted angular distribution is Rayleigh-like with depolarization ρ_Cab ≈ 0.14. Strictly, that is not isotropic, so we report false. Solvers targeting scalar RT can still project onto the m = 0 mode with a small (sub-percent) error.

source
OceanOptics.inelastic_coefficient — Function
inelastic_coefficient(p, λ_prime, λ) -> FT

Combined b^I(λ', λ) = a^I(λ') · f^I(λ', λ) [1/m/nm], the kernel of the excitation integral that an RT solver convolves against the pass-1 scalar irradiance E°(τ, λ'). Default: the product of excitation_absorption and emission. Override only if a process supplies a joint closed form cheaper than the product.
